Intro To The First Site

Lets take a quick look at what we'll be building throughout the course and the tools we need to get started

What We'll Be Building

Throughout this course we're going to learn HTML and CSS by building out the One Million Lines (make sure to click here for the link to where the OML site is now) site. In the process of making the site you will learn everything you need to know about HTML and CSS so that you can create any site you want.

I think you are going to be surprised at how easy it is to build great looking sites fast and hopefully you start feeling like you have a new super power.
